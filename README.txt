Establecer en el servidor web (MAMP, por ejemplo) la ruta de la carpeta
del proyecto.

Usando un gestor de bases de datos, crear una base de datos llamada "pw_web_app"
e importar dentro de esta el archivo .sql que se encuentra en la carpeta /assets/.

Además en /assets/functins.php debe completar los datos de la db con el host, el nombre de usuario,
la contraseña y el nombre de la db (por ejemplo en nuestro caso esos datos serían respectivamente:
'localhost', 'root', 'root', 'pw_web_app').



Entre a la ruta del proyecto mediante mediante el servidor web, y en concreto ejecute el archivo
/Mainframe/mainFrame.php
